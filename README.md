# Post-Borad
***

## :computer: Introduction
spring boot로 기본 CRUD를 구현한 익명 게시판 서버 만들기

## 설계
[API 명세서](http://documenter.getpostman.com/view/30860889/2s9YXfbNqs)

## 개발 환경
- JDK 17
- IDE : IntelliJ
- FRAMWORK : Spring Boot
- DB : MY SQL

## 주요 기능
- 포스트 작성
- 포스트 목록 조회
- 포스트 선택 조회
- 포스트 수정, 삭제 <- 비밀번호 확인
